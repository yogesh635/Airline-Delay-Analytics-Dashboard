# ✈️ Airline Delay Analytics Dashboard (Power BI)

## 🔍 Project Overview
This project is an interactive Power BI dashboard designed to analyze airline flight delays and operational performance.

It provides key insights into total flights, delayed flights, delay percentage, and average arrival delay across different routes and carriers.

---

## 🚀 Key Features
- KPI Cards (Total Flights, Delayed Flights, Delay %, Avg Arrival Delay)
- Delay Status Distribution (On-Time vs Delayed Flights)
- Origin-wise and Destination-wise Flight Analysis
- Carrier-wise Performance Comparison
- Interactive Filters (Origin, Unique Carrier)
- Detailed Table with Delay Metrics (Arrival Delay, Departure Delay, Distance)

---

## 📊 Dataset Information
- Format: Excel (.xlsx)
- Includes:
  - Origin & Destination
  - Arrival Delay
  - Departure Delay
  - Distance
  - Carrier Information

---
## 🧪 How to Test This Project

1. Download the dataset file:
   airline_data.xlsx

2. Modify the data:
   - Change delay values
   - Update origins/destinations
   - Modify carrier data

3. Open the dashboard file:
   airline_dashboard.pbix

4. Click **Refresh** in Power BI Desktop

5. Observe how dashboard visuals update dynamically
